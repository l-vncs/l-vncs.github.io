Quem sou eu?

Meu nome é Luigi Vinícius, sou técnico em informática desde 2024 e graduando em análise e desenvolvimento de sistemas (previsto para 2027). Sempre tive curiosidade pela tecnologia, desde pequeno eu já gostava de ficar no computador e nos consoles jogando e explorando o mundo tech.

Comecei a me aprofundar nessa área por volta por volta dos 14 anos, onde conheci a programação. Aprendi a fazer meus primeiros sites, ainda em um nível bem iniciante, porém não parei por aí. Atualmente estudo tanto o back-end, quanto o front-end e estou fazendo projetos para meu portfolio. Minha experiência profissional foi no âmbito do desenvolvimento web, em 2024 fiz parte da criação de um site que consistia em uma agenda virtual para escolas, era uma forma de acabar com cronogramas impressos, manter os alunos atualizados e reunir todas as informações em um único lugar. Durante esse processo de criação utilizamos diversas tecnologias como:

  • HTML;
  • CSS;
  • JavaScript;
  • PHP;
  • SQL.

    Calculadora de folha salarial (Java)

    O sistema consiste em uma tela com disponibilidade para diferentes bases salariais, podendo realizar o cálculo de salários comissionados e com bonificações. Ao final dos cadastros das informações de cada funcionário é possível imprimir uma lista com todos os nomes cadastrados.

    Para maiores informações este projeto possui o código e uma breve documentação disponível em meu GitHub: github.com/l-vncs/calculadora_salario

    Sistema de controle financeiro (Java)

    Este sistema em Java tem utilidade em controle de receitas e despesas, é possível calcular o saldo baseado nas informações cadastradas, após o término dos cadastros é possível listar o saldo. Quando o sistema recebe as saídas todas as informações são direcionadas para o banco de dados com informações como: Tipo de gasto, data e valor.

    Para maiores informações este projeto possui o código e uma breve documentação disponível em meu GitHub: github.com/l-vncs/financial-manager-java
